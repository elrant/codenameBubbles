# codenameBubbles

"Bubbles" is a messaging app that takes inspiration from Windows Live Messenger, Windows 7's Aero theme, iOS 4, Mac OS X Snow Leopard, the Skeuomorphism UI movement and Discord's voice chat and activities.
The aim of the project is to make people feel more connected with a nostalgic feeling UI that runs on the devices they're already familiar with.
